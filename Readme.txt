Pushing Docker Container into Artifact Registry (Google Cloud)

1) Enable Artifact Registry API from the console
2) Create Repository by selecting the Format (Docker) and Location

3)-----------Follow the link to create image in the google cloud artifact registry--------------
https://cloud.google.com/build/docs/build-push-docker-image

4)Deploy the container (from site)

Note: For the deployment chose 0.0.0.0:8080
